# python-rest-example
An example of building a REST app in Python.

## How to Install All Dependencies
```bash
./install.sh
```

## How to Run the App
```bash
./run.sh
```

## How to Run the Tests
```bash
tox
```

## How to View API Doc
[http://localhost:8080/swagger/index.html](http://localhost:8080/swagger/index.html)
